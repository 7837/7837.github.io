解决思路

隐式游标的名称是SQL，不能对SQL游标显式地执行OPEN,FETCH,CLOSE语句。隐式游标也有四个属性SQL%ISOPEN，SQL%FOUND，SQL%ROWCOUNT，SQL%NOTFOUND。前三种都是boolen类型，最后一个在执行任何DML语句之前，都是NULL。
例子：

输入7788显示SQL code is executed successful
